# Electronic Badge for Christmas
Herewith lies the KiCad files for an Electronic Badge made for Christmas in December 2019.

## Components used

1. OLED 128*32
1. ATTiny85
1. 0805 LEDs
1. 1206 Resistors
1. CR2032 Battery Holder
